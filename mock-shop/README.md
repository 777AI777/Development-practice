# Mock Shop (React + Vite)

1ページで商品選択から購入完了まで体験できるモックサイトです。

## 起動手順

1. `npm install`
2. `npm run dev`
3. 表示されたローカルURLを開く

## 触るファイル

- `index.html`: ページの土台
- `src/main.jsx`: 起動処理
- `src/App.jsx`: UI と状態管理
- `src/App.css`: デザインとレイアウト

## チュートリアルの狙い

- Reactの状態管理（カート、フォーム、合計金額）
- 1ページ完結の購入フロー体験
- レイアウト・アニメーション・レスポンシブ対応
