# OKINY 市場調査レポート

**調査日**: 2026年4月7日

---

## 調査方法

### 概要
Claude Code上でWebSearch機能を持つ3つのリサーチエージェントを並列起動し、日本語キーワードを中心にWeb検索を実施した。

### 調査実施日
2026年4月7日

### 調査手法
- Claude Code（Opus 4.6）のAgent機能を使用
- 各エージェントにWebSearch/WebFetchツールを付与し、日本語キーワードで検索
- 3つの調査プロンプトを並列実行（行動パターン / 競合の声 / 市場規模）

### 情報源
- Web検索結果（PR TIMES、ITmedia、note、個人ブログ、AppMatch等のレビューサイト）
- 公式プレスリリース・IR資料
- 総務省 情報通信白書
- 矢野経済研究所 推し活市場調査
- SimilarWeb推計値（非公式）

### 注意事項
- LLMの知識カットオフ（2025年8月）に基づく情報を含む。最新データは別途確認が必要
- SimilarWeb等の推計値は非公式。公式発表値と区別して記載
- ユーザーの声は検索で確認できた範囲のサンプルであり、統計的な代表性は保証しない

## 調査①: 行動パターンの存在確認

### パターン1: 「〇〇好きな人にはこれもおすすめ」投稿

- **頻度**: 映画・音楽・カフェ等で日常的に発生。映画業界がXを公式プロモーションで活用する事例（東宝・東和等）も確認
- **ハッシュタグ**: `#映画好きと繋がりたい` `#音楽好きと繋がりたい` `#本好きと繋がりたい` `#〇〇好きな人RT`
- **属性**: 趣味系コミュニティのユーザー全般。数百〜数千フォロワーの中規模アカウントが多用
- **不満**: TLに埋もれて流れる。Xのアルゴリズム変更で意図した相手に届きにくい

### パターン2: Topsters/ティアリスト画像の投稿

- **頻度**: `#私を構成する42枚`が2023年に大規模トレンド入り、2024年以降も継続。`#2022年映画ベスト10`では3,785件の有効投票（破壊屋ブログ調べ）。「ネットの音楽オタクが選ぶベストアルバム」企画は2025年で12年目、418件集計
- **ツール利用状況**:
  - **Topsters 2→3**: 音楽・映画アルバムの格子コラージュ。2024年にv2閉鎖→v3に移行。SNS連携なし、画像書き出しのみ
  - **TierMaker**: 月間1,500万〜2,000万訪問（SimilarWeb推計）。ゲームコミュニティ発祥で一般化。「投稿ボタンがあるけど使いづらいのでスクショ」というユーザー発言あり
  - **Canva**: デザイン系ユーザーがランキング・ティア表作成に利用
- **ハッシュタグ**: `#私を構成する42枚` `#私を構成する9枚` `#今年のベスト` `#2024年映画ベスト10`
- **属性**: 音楽オタク（邦楽・洋楽・インディー）が最もコア。映画ファン、ゲームコミュニティ。10〜30代中心
- **不満**: TierMakerの投稿機能が使いづらくスクショで代替。Topsters 2閉鎖でデータ喪失。画像はテキスト検索で辿れない

### パターン3: 「私のトップ3」「マイベスト」定期投稿

- **頻度**: 年末の`#今年のベスト`系で数千〜数万投稿。月次・週次で「今月の1枚」を投稿する音楽・映画アカウントも多数。音楽ナタリーがアーティストに「2024年に最も愛聴した3曲」をアンケートする企画が成立するほど定着
- **ハッシュタグ**: `#マイベスト2024` `#今年のベスト` `#2024年映画ベスト10` `#今月のベスト` `#私のベスト3`
- **属性**: 映画・音楽・読書・カフェの趣味発信アカウント。フォロワー100〜5,000の「趣味垢」が主体。20代女性が特に積極的
- **不満**: 過去の自分のベスト投稿を遡れない。プロフィール固定は1投稿のみで「好みの履歴」として機能しない

### パターン4: 「おすすめ教えて」問いかけ投稿

- **頻度**: 日常的。フォロワーが多いアカウントだとリプライが数十〜数百件規模
- **キーワード**: 「おすすめ映画教えて」「おすすめ本教えて」「カフェ教えてほしい」
- **属性**: フォロワー500〜5,000の趣味垢。回答側は同ジャンルのオタクアカウント
- **不満**: リプライで集まった情報がその投稿内で完結し、後から検索・整理できない。「教えてもらったけど忘れた」が繰り返される

### 既存ツールの限界まとめ

| ツール | 強み | 限界 |
|--------|------|------|
| Topsters 3 | 音楽・映画アルバムの格子コラージュ | SNS連携なし、画像書き出しのみ。v2閉鎖の前例 |
| TierMaker | 汎用性が高い | 投稿機能が使いづらく実質スクショ運用 |
| Filmarks | 映画記録・レビューの国内最大手 | ランキング共有・布教機能が弱い |
| Letterboxd | リスト機能充実、年間レビュー9,640万件（2024年） | 日本語対応不完全。映画以外に使えない |
| Notion | カスタム自由度が高い | 技術的ハードル高、SNS共有に不向き |

### 4パターン共通の構造的不満

1. **流れる** — TLに埋もれて後から見返せない
2. **蓄積されない** — Topsters 2閉鎖でデータ喪失の前例
3. **検索できない** — 画像投稿はテキスト検索で辿れない
4. **届かない** — Xのアルゴリズム変更で意図した相手に届きにくい

## 調査②: 競合サービスの利用者の声

### Filmarks（映画レビュー）

**良い点:**
1. 圧倒的なレビュー数とデータベース（映画・ドラマ・アニメで合計18億件超）
2. 記録の手軽さ（星だけでもOKの文化）
3. 映画発見のきっかけ（フォロー機能・レビューから）
4. SNS的な交流（映画系インフルエンサーをフォロー）
5. 多媒体対応（映画・ドラマ・アニメ）

**不満点:**
1. 映画・ドラマ・アニメ専用で他ジャンル非対応。「漫画バージョンないの？」という声
2. レビュー表示順の有料化
3. 複数回鑑賞の記録が不便
4. コメント返信機能の欠如
5. レビュー書かないと不可視化される（星のみ投稿が埋もれる）

### 読書メーター / ブクログ

**読書メーター 良い点:**
1. 専門書でもレビューが見つかる（ブクログの3.5倍のレビュー数）
2. 交流が活発（いいね、共読機能、相性機能）
3. 読書量の可視化（冊数・ページ数・著者円グラフ）
4. 複数本棚を無料で作れる

**読書メーター 不満点:**
1. レビュー文字数250字・改行不可
2. 並べ替え・検索機能が弱い
3. 本専用で他ジャンルの趣味と繋がれない
4. UIの古さ（iPhone4時代のUIという指摘）

**ブクログ 良い点:**
1. 文字数制限なし
2. 本棚UIの視認性（「書店のおすすめコーナーみたい」）
3. タグ・カテゴリ・非公開メモなど多機能
4. Amazonアソシエイト連携
5. 新刊情報・ランキングが充実

**ブクログ 不満点:**
1. ユーザー数・レビュー数が少ない
2. 交流機能が弱い
3. 読了日を登録時に入力できない
4. 読書ペースなど個人データが見づらい

### GRAVITY（やさしいSNS）

**良い点:**
1. 高い匿名性（フォロワー数非表示、実名不要）
2. 共通趣味のコミュニティ（「星」機能）
3. 心理的安全性（Xの炎上から解放）
4. エモーションテキスト・リアクション機能
5. インフルエンサーによる炎上が起きにくい

**不満点:**
1. 出会い目的ユーザーの混在
2. 匿名性ゆえの無責任投稿
3. 強制的なコンテンツ表示
4. 課金しないと使えない機能あり（広告も多い）
5. 運営対応への不満・不当な凍結

### mixi2（招待制SNS）

**良い点:**
1. 心理的安全性（招待制で信頼できる人のみ）
2. 時系列タイムライン（広告なし）
3. エモーションテキスト（エモテキ）
4. コミュニティ機能
5. 落ち着いた雰囲気

**不満点:**
1. ユーザーが急減・過疎化（2025年6月時点で廃れた印象）
2. PC版が存在しない
3. トレンド・おすすめ機能がない（新しい繋がりが生まれにくい）
4. 招待制の形骸化
5. 収益モデルが不透明（サービス継続性への不安）

### Letterboxd（海外映画SNS）

**良い点:**
1. 豊富な映画データベース
2. 詳細なスタッフ情報
3. 優れた検索・絞り込み機能
4. デザインと達成感
5. 日記形式の記録

**不満点:**
1. インターフェースが英語のみ（日本語ユーザー最大の障壁）
2. 邦画のデータが少ない
3. 日本語レビューが少ない
4. コミュニティが海外中心
5. AndroidアプリのUI問題

## 調査④: 主要SNSでの「好きなもの共有」行動分析

### Reddit

**利用実態:**
- r/Topster等でTopsters画像を共有する文化が定着。r/Music（2,770万メンバー）等でマイベスト投稿
- TierMakerと連携したティアリスト画像投稿も活発（upvote 1,000〜10,000が一般的）
- 日本は先進国中Reddit利用率が最低水準。日本語コミュニティは極小

**良い点:**
1. コミュニティの専門性・深度が高い（ジャンル別サブレ）
2. upvoteによる民主的な品質フィルタリング
3. Google経由での検索・蓄積性（「site:reddit.com best」で過去の良質リストが見つかる）
4. コメント文化による深い対話（「なぜこれを選んだか」）
5. 英語圏なら大規模な共感コミュニティ

**不満点・限界:**
1. 日本語コミュニティが極小（先進国最低水準の利用率）
2. UIの学習コストが高い（サブレ選択・karma・投稿ルール）
3. time decayアルゴリズムで古い投稿が埋もれる
4. 「自分の好き」を整理・管理するツールではない（フォーラム構造）
5. モバイルアプリへの不満が根強い

**OKINYとの差別化:**

| 軸 | Reddit | OKINY |
|---|---|---|
| 目的 | フォーラムでの議論・情報共有 | 「好き」の整理・保存・共有に特化 |
| 構造 | 投稿は時系列で流れる | リスト形式で永続的に蓄積 |
| 言語 | 英語圏中心 | 日本語ネイティブ |
| 参入障壁 | 高い（karma・ルール等） | シンプル（リスト作るだけ） |
| 検索性 | 外部Google依存 | タグ検索で体系的に発見 |

**ユーザー規模:**
- グローバルMAU: 約13.6億人（2026年見込み）
- グローバルDAU: 約1.214億人（Q4 2025）
- 日本: 具体的データなし、推定数十万人規模

### Threads

**利用実態:**
- テキスト中心（最大500文字）＋最大10枚の画像。カルーセル投稿が日本で特に人気
- 「○○ランキング形式」「マイベスト○選」がバズりやすい形式として認知
- 2024年10月から日本でトレンドランキング機能が先行テスト開始（米国以外初）
- リプライへの参加時間がTL閲覧時間の約半分 → 「おすすめ教えて」型が反応取りやすい

**良い点:**
1. Instagramとの連携で初期フォロワーが即獲得できる
2. テキスト＋画像の柔軟な組み合わせ（500文字＋画像/動画）
3. エンゲージメント率が高い（平均4.51〜6.25%、Instagramより高い）
4. 若年層・女性ユーザーが多い（20代24.8%、女性53.8%）
5. Instagram Storiesへのワンクリック展開でリーチ拡張

**不満点・限界:**
1. 検索機能が弱い（ハッシュタグ1つのみ、過去投稿の発見精度が低い）
2. タイムライン型でコンテンツが流れる（ポートフォリオ的蓄積ができない）
3. 軽度ユーザーが多い（日本ユーザーの57%が月1〜5日しか使わない）
4. 独自の価値提案が不明確（「テキストならX」「写真ならInstagram」との差別化が弱い）
5. アルゴリズムが不透明（フォロー外へのリーチ戦略が立てにくい）

**OKINYとの差別化:**

| 軸 | Threads | OKINY |
|---|---|---|
| コンテンツ構造 | 時系列タイムライン（流れる） | リスト形式で永続保存 |
| 検索・発見 | ハッシュタグ1つ・検索弱い | タグ検索・体系的発見 |
| 目的 | 日常会話・カジュアル発信 | 「好き」の整理・記録・共有に特化 |
| コンテンツの深度 | 1投稿で完結（薄い） | リスト構造で「なぜ好きか」が伝わる |
| 蓄積性 | ほぼなし | プロフィールに「好きの辞書」として蓄積 |

**ユーザー規模:**
- グローバルMAU: 4億人（2025年8月）
- グローバルDAU: 1億人超（2024年12月）
- 日本: 1,230万人（2025年8月、2年間で222%成長）
- 日本ユーザー属性: 女性53.8%、男性46.2%、20代が最大層

### Instagram

**利用実態:**
- カルーセル投稿で「マイベスト○○」「ランキング」を共有する文化が活発（2025年にアルゴリズムがカルーセル優遇に変化）
- ストーリーズのインタラクティブ機能（質問・アンケート・クイズスタンプ）で「好きなもの」のやりとりが日常化。日本のDAUの70%がストーリーズ使用
- ハイライト機能で「好きな映画まとめ」「行ったカフェ」等のカテゴリ別アーカイブを作る使い方が定着（2025年9月にグリッド表示・並べ替え対応）
- ジャンル特化アカウント文化: `#映画好きな人と繋がりたい` `#カフェスタグラム` `#読書記録` `#bookstagram`
- 日本ユーザーはハッシュタグ使用頻度が他国平均の5倍
- 2025年にかけて「インスタ映え」一辺倒から「リアルな日常・本音」へ移行傾向

**良い点:**
1. ビジュアル訴求力が圧倒的（映画ポスター・カフェ写真・本の表紙を一瞬で伝えられる）
2. ハイライトによる恒久的蓄積（2025年9月アップデートでグリッド化・並び替え対応）
3. 発見タブによるアルゴリズム推薦（興味・関心ベースで同じ「好き」のアカウントを推薦）
4. ストーリーズのインタラクティブ性（質問・アンケート・クイズ）
5. ユーザー規模が国内最大級（約6,600万人、女性・若年層中心）

**不満点・限界:**
1. テキスト検索が弱い（「この人の好きな映画を全部見たい」ができない）
2. ジャンル横断がしにくい（アカウントのテーマ縛り。映画・音楽・カフェを混ぜるとテーマが崩れる）
3. 映え疲れ・ビジュアル品質のプレッシャー（Z世代の51%がSNS疲れ）
4. 構造化・ランキング化が不得意（カルーセルで見せられるが、データとして残らない）
5. コンテンツの流動性（ストーリーズは24時間で消える。ハイライトは手動管理が必要）

**OKINYとの差別化:**

| 軸 | Instagram | OKINY |
|---|---|---|
| 情報形式 | ビジュアル主体 | テキスト＋順位という構造データ |
| 蓄積方法 | ハイライトに手動保存 | リスト形式で自動構造化 |
| ジャンル横断 | アカウントのテーマ縛り | 1ユーザーが複数ジャンルのリストを持てる |
| 検索性 | ハッシュタグ検索（弱体化中） | タグ＋テキストで構造的に検索可能 |
| 参入コスト | 「映え」のビジュアル要求あり | テキストだけで投稿できる低コスト |

**ユーザー規模:**
- 国内MAU: 約4,230万〜6,600万人（調査機関により差異）
- 国内ランキング: 3位（1位LINE、2位YouTube）
- 全年代利用率: 48.5%、20代利用率: 78.6%
- 男女比: 女性54.8% / 男性42.3%
- ストーリーズDAU: ユーザーの70%が毎日使用
- 世界MAU: 約20億人超（2025年）

### X（旧Twitter）

**利用実態:**
- 「布教」文化が最も根付いたSNS。映画・音楽・本・アニメの感想ポストが大量に流れる
- コミュニティ機能（2025年2月にメンバー外にも公開・返信可能に変更）で「映画好き」「読書好き」等のテーマグループが多数存在。ただしリアルタイム会話・実況が主体
- ブックマーク: フォルダ分け・タグ付け・テキスト検索なし（プレミアムでフォルダのみ）。大量保存すると「情報の沼」化
- リスト機能: 「人を整理する」機能であって「コンテンツを整理する」機能ではない

**良い点:**
1. テキストの拡散力（「おすすめ10選」がリツイートで爆発的に拡散）
2. リアルタイム性・共感の連鎖（同じ映画を見た人が同時に反応する体験）
3. テキスト検索（Instagramよりはキーワードで過去ポストを探せる）
4. コミュニティ機能（2025年2月以降は公開性が上がり発見されやすい）
5. 匿名・低コスト投稿（ビジュアル不要、テキストだけで気軽に発信）

**不満点・限界:**
1. 投稿の流動性が致命的（関連性スコアが6時間ごとに50%減少。過去の「好き」を見返せない）
2. 構造化・蓄積が不可能（ブックマークも検索・タグなしの時系列リスト）
3. 「おすすめ汚染」問題（非フォロー投稿の混入、広告、政治コンテンツ。Bluesky移行の主因）
4. 有料化・API制限（2024年11月にBasicプラン倍額$100→$200、情報収集bot多数停止）
5. 信頼性・安全性の懸念（ヘイト・偽情報対策の後退。日本でも利用率前年比-5.7pt）

**OKINYとの差別化:**

| 軸 | X | OKINY |
|---|---|---|
| 情報の流動性 | 高（=蓄積されない） | 低（=整理・保存される） |
| 構造化 | 不可（フラットなポスト） | リスト形式で構造化 |
| 蓄積・検索性 | 弱（ブックマークは沼化） | タグ・タイトルで後から参照可能 |
| 「好き」の全体像 | 見えない（時系列に埋もれる） | プロフィール上に全ジャンルのリストが並ぶ |
| 安心して語れる場 | 政治・炎上リスクあり | 趣味特化で純粋に好きを語れる |

- 国内MAU: 約6,700万〜6,928万人（2025年11月推定）
- 国内ランキング: 2位（1位LINE）

- 男女比: ほぼ1:1（やや男性多め）

## OKINYへの示唆（総合分析）

### 1. クロスジャンルの空白地帯

映画・本・音楽・カフェを1箇所でまとめたいニーズは確実に存在。Filmarks・読書メーター・Letterboxdのユーザーが複数アプリを掛け持ちしている実態が確認された。現状すべてのサービスが単一ジャンル特化であり、ここに明確な空白がある。

### 2. 「並べる」は「書く」より手軽

レビューを書くのは面倒だが、順位をつけて並べるならハードルが低い。Topsters/TierMakerの利用実態がこれを裏付ける。OKINYのリスト形式は「レビュー疲れ」への解答になりうる。

### 3. 蓄積性の欠如

Xは拡散に強いが蓄積に弱い。「自分の好みの履歴」として機能するサービスが市場に存在しない。Topsters 2閉鎖によるデータ喪失は象徴的事例。

### 4. 心理的安全性の需要

X上の議論・炎上を避けたい層がGRAVITY・mixi2に流れているが、GRAVITYは出会い目的混在、mixi2は過疎化という課題を抱える。「好き」に特化した安全な場の需要がある。

### 5. ニッチではない市場規模

趣味目的SNS利用が20〜30代の過半数、推し活市場は7,000億円規模。Letterboxdが映画特化で1,000万ユーザーを達成した成長軌道は、クロスジャンル展開のポテンシャルを示唆。
